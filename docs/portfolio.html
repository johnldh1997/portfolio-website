Mobile App · Transport

Auckland Transit

A personal, from-scratch alternative to Auckland Transport’s official app — features live vehicle tracking, journey planning with real-time GPS step tracking, background-surviving trip notifications, and an offline-first schedule database.

  • Built a compressed 58MB on-device SQLite database from AT's static GTFS feed, reducing raw GTFS data size by 64% to enable offline schedule querying
  • Implemented GPS-based step matching with schedule fallback and an Android foreground notification system that survives app termination
  • Engineered custom map staleness fixes and synchronous OTA update checks via EAS Update for real-time tracking and continuous deployment
